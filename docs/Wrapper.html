Wrapper ( string dir, object commands )

Wrapper includes all core and custom modules.

See Wrapper

Parameters

string dir

The absolute directory path to the project.
object commands

An object with commands for the webcrawler.

string startUrl = https://www.phpdoc.org/

The url you are starting from. (Optional)
number pageLimit = 0

The limit of pages to crawl. 0 will be infinite. (Optional)
number debug = 0

Turn on/off the debug mode. (Optional)
string debugModule=

Specify a module to debug. If empty and debug is on, all modules will be debugged. (Optional)
number screenShots = 0

Turn on/of the function to create screenshots. (Optional)

Return Value

Wrapper

Returns the Wrapper instance.
